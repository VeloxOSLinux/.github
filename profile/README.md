**The Speed of CachyOS meets the Stability of Manjaro.**

VeloxOS is a performance-optimized Linux distribution designed for modern hardware. By leveraging the **CachyOS Kernel** and **x86-64-v3/v4** instruction sets, we provide a smooth, responsive, and gaming-ready desktop experience based on the reliable Arch-Manjaro ecosystem.

## üõ† Core Features

- **Turbocharged Kernel:** Powered by the CachyOS Kernel (BORE scheduler) for ultra-low latency.
- **Modern Optimizations:** Compiled for x86-64-v3 and v4 to unlock the full potential of your CPU.
- **Gaming Ready:** Pre-configured tools and drivers for an out-of-the-box performance boost.
- **Reliable Base:** Built upon Manjaro's stable core for a rock-solid daily driver.
